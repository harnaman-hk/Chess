pawn module¶

This module contains functions to handle all the pawn moves in the game, namely:

  • Ordinary Pawn moves
  • Pawn Captures
  • Pawn promotions
  • En Passant
pawn.capture(move, board_view, piece_view)¶

Updates the board with a pawn capture and deletes the captured piece

pawn.is_capture(move)¶

return type: bool returns True if the move is a pawn capture

pawn.is_enpassant(move)¶

return type: bool returns True if the move is an En Passant

pawn.is_promotion(move)¶

return type: bool returns True if the move is an pawn promotion

pawn.is_regular_pawn_move(move)¶

return type: bool returns True if the move is an ordinary pawn move

pawn.move_pawn(move, board_view, piece_view)¶

Updates the board with a regular pawn move

pawn.promote(move, board_view, piece_view)¶

This function handles a pawn promotion move.
