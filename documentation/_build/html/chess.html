Main module¶

Pythonic Approach to play Chess¶

  • Pre-process the .pgn file to get obtain only the move of the game
  • Read each move and update the board. Display the updated board if the user wishes so.
  • Handle exceptional cases like En Passant, Castling, Capture, etc.
  • Display the final position of all the pieces on the board.
chess.castle(move, board_view, piece_view)¶

This function performs the castling move, both the king side casting and queen side castling. An updated version of the board.

chess.get_from_square(move, board_view, candidates)¶
Parameters:
  • move – move in play
  • board_view – position of pieces on board

return type: str

returns the initial position of a piece given its final position

chess.is_capture(move)¶

return type: bool

returns True if the move in play is a capture move

chess.is_castling(move)¶

return type: bool

returns True if the move in play is a Castling move (King side or Queen side Castling)

chess.is_pawn_move(move)¶

return type: bool

returns True if the move in play is a pawn move

chess.make_move(piece, move_from, move_to)¶
Parameters:
  • piece – chess piece to be moved in the current move
  • move_from – initial position of piece
  • move_to – final position of piece

Update the board and piece statistics with the current move in play

chess.not_blocked(board_view, a, b)¶

return type: bool returns True if there is no obstruction on board between a and b

chess.process_moves(moves)¶

This function breaks each sequence in the list of all_moves into black or white move depending on the course of game. It then passes the respective moves to make_move() for the moves to be updated on the board.

chess.setup()¶

Initialises the chess board to default state before the game begins
