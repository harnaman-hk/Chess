PgnParser module¶

This module includes functions to clean and pre-process the input .pgn file

PgnParser.clean(move)¶

return type: str Each move is filtered to remove any data other than Alpha-numeric characters

PgnParser.pgn_to_moves(game_file)¶

Reads a .pgn file. Further, beggining remarks, comments, indexing, etc are removed from the input so that only the moves processed further.

PgnParser.pre_process_a_move(move)¶

return type: str add ‘P’ for pawn move capital notations for white pieces, small for black

PgnParser.pre_process_last_move(move)¶

return type: str As the last move may contain extra information regarding the result of a match, it is handled separately. All data except the move is removed.

PgnParser.pre_process_moves(moves)¶

return type: list[ str ] returns a list of sequence of black, white moves in a game
